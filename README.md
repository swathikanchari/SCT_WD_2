# SkillCraft Technology Internship - Task 2

## Project Title

Stopwatch Web Application

## Description

This project is a Stopwatch Web Application developed as part of the SkillCraft Technology Web Development Internship.

The stopwatch application includes features like Start, Pause, Reset, and Lap time recording to provide accurate time tracking with a simple and interactive user interface.

## Technologies Used

* HTML
* CSS
* JavaScript

The application is fully responsive and works smoothly on different screen sizes and devices.

## Features

* Start Stopwatch
* Pause Stopwatch
* Reset Timer
* Record Lap Times
* Responsive Design
* Interactive User Interface
